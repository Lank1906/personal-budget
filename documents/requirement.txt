1. INTRODUCTION
1.1 System Objectives
The Personal Budget app aims to help users manage their personal finances with ease. Primary objectives include:

- Track daily income and expenses
- Set budgets by month and by category
- Analyze and report financial status
- Provide smart suggestions to save money and optimize spending

1.2 Target Users
- Working professionals seeking to manage their personal expenses
- Students who want to control allowances or scholarship money
- Small families who need to manage monthly household budgets

2. PROJECT SCOPE
The application will be developed as a mobile app (iOS/Android) with the following core features:

- User registration and login
- Add income and expense transactions
- Assign categories, notes, and timestamps
- Create and monitor budgets
- View financial charts and reports
- Receive smart financial suggestions based on user data
